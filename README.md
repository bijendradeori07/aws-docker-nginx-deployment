# AWS Docker Nginx Deployment
## Project Overview
This Project demonstrate deploying a containerized Nginx web server on AWS EC2 using Docker
## Technologies used
- AWS EC2
- Docker
- Nginx
- Ubuntu Linux
## Architecture
User - Internet - AWS EC2 - Docker Container - Nginx Web Server
## Steps Performed
1. Created EC2 instance
2. Installed Docker
3. Pulled Nginx image
4. Ran container with port 80
5. Accessed web server through public IP
## Result
Nginx web server successfully deployed and accessible through browser.
